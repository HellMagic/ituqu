###what's this?
途趣网是一个为驴友提供服务的网站，目前还没有什么特色功能，[途趣网](http://www.tuqu365.com/)。

###网站的构想：
基于地点的信息分享服务：
   1. 分享东西时，提供选择地点的功能。
   2. 通过地点进行搜索。
   3. 建立一个旅游方面的资料库。
